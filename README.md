## 配置bash和git
配置 .bash_profie
```
alias ..="cd .."
alias cl="clear"
alias gi="git push"
alias gl="git pull"
```
配置 git 用户名
```
git config --global --edit
git config --global user.name "bob"
git config --global user.email bob@gmail.com
git config --global init.defaultBranch master
```
