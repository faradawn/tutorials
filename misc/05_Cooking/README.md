# 做饭手册

## Part 1 - 菜谱
### 滑溜肉片 / 土豆丝肉片
视频
- [不用研制](https://www.bilibili.com/video/BV1FJ411u7pM/?spm_id_from=333.788.recommend_more_video.-1)

备料
- 化冻水龙头滴水，20min；开水10min但会变白
- 切肉片，可用刀背面拍打成片
- 放碗里倒入料酒，盐，淀粉，拌匀，研制10min
- 胡萝卜斜着片，黄瓜片
- 姜和大葱

制作  
- 烧锅，倒入油20s，炒肉片20s
- 姜和大葱，炒一下
- 土豆丝，炒三下
- 生抽
